    üö® Error Handling & Debugging di Python

    Error pasti terjadi dalam coding, tapi jangan panik! Python punya mekanisme error handling biar program gak langsung crash. Kita juga bakal belajar debugging buat jadi detektif kode yang handal. üîçüïµÔ∏è‚Äç‚ôÇÔ∏è

    ‚ö†Ô∏è 1. Try-Except-Finally (Menangani Error dengan Elegan)

    try-except digunakan untuk menangkap error tanpa bikin program berhenti.

    üîπ Contoh Sederhana

    try:
    angka = int(input("Masukkan angka: "))
    hasil = 10 / angka
    print(f"Hasil: {hasil}")

except ZeroDivisionError:
    print("Error! Tidak bisa membagi dengan nol.")

except ValueError:
    print("Error! Masukkan angka yang benar.")

except Exception as e:
    print(f"Terjadi error: {e}")

finally:
    print("Program selesai.")

    üìå finally akan selalu dijalankan, entah ada error atau tidak.
    üìå Exception as e menangkap semua jenis error.

    Contoh Output: 

    Masukkan angka: 0
Error! Tidak bisa membagi dengan nol.
Program selesai.

    üöÄ 2. Raise & Custom Exception

    Kita juga bisa membuat error sendiri dengan raise.

    üîπ Raise Exception Manual

    def cek_umur(umur):
    if umur < 18:
        raise ValueError("Umur harus minimal 18 tahun!")
    return "Akses diberikan."

try:
    print(cek_umur(16))
except ValueError as e:
    print(f"Error: {e}")

    üìå Hasilnya: 

    Error: Umur harus minimal 18 tahun!

    üîπ Membuat Exception Sendiri

    class CustomError(Exception):
    pass  # Bisa ditambah fitur sendiri

try:
    raise CustomError("Ini error khusus!")
except CustomError as e:
    print(f"Custom Error: {e}")

    üïµÔ∏è‚Äç‚ôÇÔ∏è 3. Debugging Kayak Detektif

    Debugging adalah proses mencari & memperbaiki bug dalam kode. Python punya beberapa tools keren buat ini!

    üîπ 1. Gunakan Print Statement üì¢

    Kadang, print() bisa bantu cari bug. 

    def bagi(a, b):
    print(f"DEBUG: a={a}, b={b}")  # Debugging pakai print
    return a / b

print(bagi(10, 2))

    üîπ 2. Gunakan assert Buat Validasi

    def cek_nilai(n):
    assert n > 0, "Nilai harus lebih besar dari 0!"
    return f"Nilai valid: {n}"

print(cek_nilai(5))  # Berhasil
print(cek_nilai(-3))  # AssertionError

    üîπ 3. Debugging Pakai pdb (Python Debugger) üïµÔ∏è‚Äç‚ôÇÔ∏è

    Ketik import pdb; pdb.set_trace() di mana saja buat masuk mode debugging interaktif. 

    import pdb

def hitung(a, b):
    pdb.set_trace()  # Debugging mulai di sini
    return a + b

hitung(10, 5)

    üìå Command Debugging di pdb:
    - n (next) ‚Üí Jalankan kode baris demi baris
    - c (continue) ‚Üí Lanjutkan eksekusi normal
    - p variabel ‚Üí Cetak nilai variabel

    üéØ Kesimpulan

    ‚úÖ Try-Except-Finally ‚Üí Tangkap error tanpa bikin program crash.
    ‚úÖ Raise & Custom Exception ‚Üí Bisa bikin error sendiri sesuai kebutuhan.
    ‚úÖ Debugging ‚Üí Bisa pakai print, assert, atau Python Debugger (pdb).
