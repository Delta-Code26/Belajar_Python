    üìä Python untuk Data Science

    Python adalah bahasa utama dalam Data Science karena memiliki pustaka yang kuat untuk analisis data, visualisasi, dan machine learning seperti Pandas, NumPy, Matplotlib, dan Seaborn.

    üîß 1. Instalasi & Setup Data Science di Python

    üìå Install pustaka yang dibutuhkan 

    pip install numpy pandas matplotlib seaborn

    ‚úÖ Sekarang kita siap menganalisis data!

    üìÇ 2. Baca & Manipulasi Data dengan Pandas

    üìå Baca dataset CSV dan tampilkan informasi dasar 

    import pandas as pd

# Load dataset
df = pd.read_csv("data.csv")

# Tampilkan 5 baris pertama
print(df.head())

# Cek info dataset
print(df.info())
print(df.describe())

    ‚úÖ Gunakan Pandas untuk mengolah data dengan mudah!

    üìå Menghapus kolom atau baris yang memiliki nilai kosong 

    df.dropna(inplace=True)  # Hapus baris dengan nilai kosong
df.drop(columns=["KolomTidakPenting"], inplace=True)  # Hapus kolom tertentu

    ‚úÖ Bersihkan data sebelum dianalisis!

    üìà 3. Visualisasi Data dengan Matplotlib & Seaborn

    üìå Buat grafik sederhana untuk memahami data 

    import matplotlib.pyplot as plt

# Buat grafik batang
df["Kategori"].value_counts().plot(kind="bar", color="skyblue")
plt.title("Distribusi Kategori")
plt.xlabel("Kategori")
plt.ylabel("Jumlah")
plt.show()

    ‚úÖ Bisa memahami tren data dengan cepat!

    üìå Gunakan Seaborn untuk visualisasi yang lebih menarik 

    import seaborn as sns

# Scatter plot hubungan antara dua variabel
sns.scatterplot(x="Pendapatan", y="Pengeluaran", data=df)
plt.title("Hubungan Pendapatan vs Pengeluaran")
plt.show()

    ‚úÖ Bisa melihat pola dalam dataset dengan lebih jelas!

    üìä 4. Analisis Statistik Data

    üìå Hitung korelasi antar variabel untuk menemukan hubungan dalam data 

    print(df.corr())  # Hitung korelasi antar kolom

    üìå Visualisasi korelasi dengan heatmap 

    sns.heatmap(df.corr(), annot=True, cmap="coolwarm")
plt.title("Heatmap Korelasi")
plt.show()

    ‚úÖ Mengetahui hubungan antar variabel sebelum analisis lebih lanjut!
